Landing Page – Rock Band: Shattered Riffs 

Project Description 

This project consists of the development of a landing page for a rock band called Shattered Riffs, built using HTML and CSS. The page features key information such as upcoming events, band biography, and multimedia content, all wrapped in a modern design that is fully responsive across different devices. 

Objectives 

Properly implement HTML semantic tags. 

Apply styling using CSS (Flexbox, Grid, and Media Queries). 

Develop a responsive and visually cohesive layout. 

Project Structure 

Plaintext 

/project 
│── index.html 
│── style.css 
│── README.md 
 

 programs Used 

HTML5: Site structure. 

CSS3: Styling and layout. 

 

📄 HTML Structure 

The index.html file is organized semantically to ensure accessibility and SEO best practices: 

🔹 Header 

Contains the brand identity and navigation: 

Band Logo. 

Navigation Menu. 

HTML 

<header> 
   <h2>Shattered Riffs</h2> 
   <nav>...</nav> 
</header> 
 

🔹 Hero Section 

The main "above the fold" section: 

Catchy headline. 

Short description. 

Call to Action (CTA) button. 

HTML 

<section class="hero"> 
   <h1>Rock the Night</h1> 
   <a href="#" class="btn">View Events</a> 
</section> 
 

 

🔹 Events Section 

An HTML table is used to display the tour dates: 

<thead> for column headers. 

<tbody> for the event details. 

HTML 

<table> 
   <thead>...</thead> 
   <tbody>...</tbody> 
</table> 
 

🔹 "About the Band" Section 

Includes: 

Band biography/description. 

An unordered list (<ul>) featuring band members or highlights. 

🔹 Multimedia Section 

Implemented using CSS Grid to organize an image gallery effectively. 

🔹 Footer 

Contains basic site information, copyright, and social links. 

 

🎨 CSS Styles 

The style.css file manages the visual identity. It starts with a global reset to ensure consistency across browsers: 

CSS 

* { 
   margin: 0; 
   padding: 0; 
   box-sizing: border-box; 
}
